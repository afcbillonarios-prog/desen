Crédito de Fomento Agropecuario

Aplicación del Ciclo de Vida del Machine Learning (CRISP-ML) para la predicción inteligente del valor de desembolsos crediticios.

Conocer el Modelo

Identificación del Problema

El acceso al financiamiento es crucial para el desarrollo del sector agropecuario. Sin embargo, determinar el monto ideal del crédito (valor de crédito) que se debe desembolsar a un productor dependiendo de las condiciones financieras de la solicitud (plazos, cuotas, tasas) es un desafío. El problema radica en estandarizar y predecir este valor basándose en el comportamiento histórico de los desembolsos.

Propósitos y Objetivos

  • Objetivo General: Desarrollar un modelo de Machine Learning capaz de predecir el valor del crédito agropecuario utilizando la metodología CRISP-ML(Q).
  • Propósitos Específicos:
    • Realizar un proceso de ETL estructurado para limpiar los datos históricos.
    • Ejecutar un EDA para encontrar relaciones clave entre las variables.
    • Entrenar y evaluar un modelo de Regresión Lineal.
    • Desplegar el modelo de forma interactiva usando Streamlit.

Tabla de Datos y Variables

El modelo utiliza las siguientes variables predictoras extraídas de la muestra del conjunto de datos original:

Variable Tipo Descripción
plazo Numérico Plazo total del crédito en meses.
periodo_gracia Numérico Meses de gracia otorgados antes de iniciar el pago a capital.
cuotas Numérico Número total de cuotas pactadas.
cuotas_capital Numérico Número de cuotas destinadas exclusivamente a abonar capital.
tasa_indexacion Categórico Indicador de la tasa base (ej. IBR, DTF). Se aplica One-Hot Encoding.
valor_credito Numérico (Variable Objetivo) El valor desembolsado del crédito.
